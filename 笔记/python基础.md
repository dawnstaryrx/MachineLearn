# python基础

## 1 谈谈你对python语言的认识？

1. Python是一种解释型语言，这意味着在开发过程中没有了编译这个环节。
2. Python是一种交互式语言。
3. Python是面向对象的语言，支持面向对象的风格或代码封装在对象的编程技术。
4. Python是开源的，拥有很庞大的社区。
